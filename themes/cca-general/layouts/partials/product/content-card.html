Storage

Cloud storage is a key strategic component for cloud deployments requiring data sovereignty or embracing a data trust philosophy. Towards this goal, cloud.ca provides a variety of storage offerings, all with the guarantee of high-security, high-availability and that data will always remain in Canada.

Performance Block Storage with QoS

This all SSD-driven primary storage provides 3 tiers of guaranteed performance to meet the needs of any critical, performance-oriented workload

Ideal uses: Latency sensitive applications, large SQL/NoSQL databases, AI

Performance Block Storage (no QoS)

This hybrid-SSD primary storage provides an ideal price/performance balance for workloads with more flexible performance requirements

Ideal uses: Web server, applications server, databases

Object storage

This storage tier provides highly-scalable, flexible Swift and S3 compatible redundant object storage that is ideal for inline application usage as well backup and archival. Storage policies provide for both single-region and multi-region capabilities

Ideal uses: Disaster recovery — Backup and Archiving, Static web site hosting and static content distribution, Document Store and file sharing, Big Data analytics

Dedicated Infrastructure

While there are many benefits to multi-tenant cloud infrastructure, the cloud.ca team recognizes that this doesn’t solve all customer needs, particularly for certain privacy and security oriented use cases. This is why we offer flexible dedicated infrastructure options.

Dedicated infrastructure in cloud.ca can be provisioned to meet specific customer requirements, while still providing the benefit of predictable monthly Opex costs.

A number of options are available, including:

  • Dedicated compute clusters
  • Dedicated storage
  • Dedicated networking/security

Each of these can be deployed seamlessly with complementary cloud.ca multi-tenant infrastructure.
