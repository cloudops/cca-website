Storage

Cloud storage is a key strategic component for cloud deployments requiring data sovereignty or embracing a data trust philosophy. Towards this goal, cloud.ca provides a variety of storage offerings, all with the guarantee of high-security, high-availability and that data will always remain in Canada.

Performance Block Storage with QoS
This all SSD-driven primary storage provides 3 tiers of guaranteed performance to meet the needs of any critical, performance-oriented workload
Ideal uses: Latency sensitive applications, large SQL/NoSQL databases, AI
Performance Block Storage (no QoS)
This hybrid-SSD primary storage provides an ideal price/performance balance for workloads with more flexible performance requirements
Ideal uses: Web server, applications server, databases
