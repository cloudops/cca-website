BUILD REQUIREMENTS
==================

Install Dependencies
--------------------

*Make sure you have `hugo` version `v.41` or higher*

**MacOS or Linux**

Make sure `npm` is installed on your system.

``` bash
$ npm install less -g
$ npm install uglifycss -g
$ npm install uglify-js -g
```


Build Project
-------------

``` bash
$ make build
```


Run Project
-----------

``` bash
$ make run
```
